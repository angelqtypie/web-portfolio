## About the Portfolio Website

This website is my **personal portfolio**, showcasing my skills, projects, and journey as a **BSIT-Student/Web Systems Developer**.  

It includes:

- **Navigation:** Smooth scrolling navbar for Home, Projects, Tech Stack, and Contact.
- **Hero Section:** Short introduction with profile image and description.
- **Projects Section:** Highlights projects with live links.
- **Tech Stack:** Displays the technologies and tools I use.
- **Journey Section:** Shows my educational path and my areas of interest.
- **Contact Section:** Visitors can send messages via a secure EmailJS form.

---

## Contact Form & EmailJS Integration

### How It Works
1. Users submit **name, email, and message**.
2. EmailJS delivers the message directly to my inbox.

### Security of the Public Key
- The **public key** is safe for frontend use and cannot access sensitive backend data.
- EmailJS validates **Service ID** and **Template ID**, so exposing the public key does **not allow misuse**.
- **Extra security:** Domain restrictions are enabled in EmailJS to allow requests **only from my deployed domain**.
